HTML и базовая структура

Основная структура HTML документа выглядит следующим образом:

                
    <!DOCTYPE html>
    <html lang="en">
    <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    </head>
    <body>
    </body>
    </html>

Тег <!DOCTYPE html> определяет тип документа html, а тег <html lang="en"> определяет язык en.

Тег <meta charset="UTF-8"> определяет кодировку символов в HTML документе.

Тег <meta name="viewport" content="width=device-width, initial-scale=1.0"> определяет размер окна браузера.

Тег <title>Document</title> определяет заголовок документа.

Тег </head> определяет заголовок документа.

Тег <body> определяет содержание документа.

Тег </html> определяет конец документа.

Существует еще тег <link/> и <script/>.

Тег <link/> это тег , который позволяет связывать документ HTMl с CSS документом, в котором хранятся стили.

Также этот тег позволяет добавлять различные шрифты в документ.

Тег <script/> это тег , который позволяет связывать документ HTMl с JavaScript документом.

    Стили (CSS)

    Чтобы как-либо преобразовать текст или что-нибудь другое в HTML документе, можно использовать стили CSS.

    Разберем пример смены текста у тега <p>

                    
    HTML:
    <p>Hello, World!</p> //Чтобы сменить текст в теге <p> нам потребуется атрибут class
    <p class="text-main">Hello, World!</p> // задаем класс и теперь в css документе можно использовать этот класс и сменить цвет
    CSS:
    .text-main {
        color: blue;
    } //Готово,теперь цвет текста будет синим

    Hello, World!

    Допустим, мы хотим поменять размер шрифта, но мы не знаем как это сделать. Вот теперь пример смены размера текста:

                    
    HTML:
    <p class='font-change'>Hello, World!</p> //Создаем класс font-change
    CSS:
    .font-change {
        font-size: 24px; //Задаем размер шрифта с помощью font-size : 24px; //24px - это пиксели(в нашем случае 24 пикселя)
    }

    Hello, World!

    Первым делом, нужно зайти на Google Fonts и найти шрифт, который нам понравится.

    Далее выбираем радиокнопку с надписью <link> и снизу в окошке появляется код

    Мы копируем этот код и вставляем в HTML документ(в теге <link>, который должен находиться в теге <head>)

    Теперь мы начнём менять шрифт

                    
    HTML:
    <p class='another-font'>Hello, World!</p>
    CSS:

    Стили для контейнеров:

    Все эти стили можно применять практически к любому тегу(например, <div>, <footer> и т.д.)

    Вернемся к display:flex

    Представим ситуацию, когда мы хотим в нашем контейнере поместить все элементы в строчку или столбик

    Вот способ реализации этого:

                    
    HTML:
    <div class='flex'>
        <p>Hello, World!</p>
        <a>Hello, World!</a>
    </div>
    CSS:
    .flex {
        display: flex;
        align-items: center;
        flex-direction: column; //наши элементы будут выстроены в столбик
        // есть другое значение: row; //наши элементы будут выстроены в строчку
        // есть другое значение: row-reverse; //наши элементы будут выстроены в строчку в обратном порядке
        // есть другое значение: column-reverse; //наши элементы будут выстроены в столбик в обратном порядке
    }

    Как создать красивую рамку для контейнера и сделать ее закругленной?

                    
    HTML:
    <div class='border'>
        <p>Hello, World!</p>
        <a>Hello, World!</a>
    </div>
    CSS:
    .border {
        border: 1px solid red; //Делаем рамку , которая будет иметь высоту 1px и покрасим её в красный, а параметр solid рисует рамку сплошной линией
        border-radius: 5px; //Закругляем рамку на 5px 
    }

    Результат:

    Hello, World!

    Hello, World!

    Текст немного смещен в рамку, чтобы это исправить можно использовать padding или margin

    В чем же их различие?Падинги (свойство padding ) — это внутренние отступы от границ элемента до его содержимого. Маржины (свойство margin ) — это внешние отступы от границ элемента до соседних элементов

    Сделаем отступы для текста

                    
    HTML:
    <div class='border'>
        <p>Hello, World!</p>
        <a>Hello, World!</a>
    </div>
    CSS:
    .border {
        padding: 5px; //внутренние отступы от границ элемента до его содержимого
        margin: 5px; //внешние отступы от границ элемента до соседних элементов
        border: 1px solid red;
        border-radius: 5px;
    }

    Результат:

    Hello, World!

    Hello, World!
